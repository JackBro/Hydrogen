This is the Hydrogen Debugger. This allows programmers to trace their programs and see in detail how they run. It is modeled after gdb and lldb and designed to work with the Hydrogen virtual machine.

To start hdb use:

./hdb <program name>

To add a breakpoint, use:
brk <line number>


To print the value of a variable:
prnt <variable name>

To run the program use:
run

To quit hdb:
quit
