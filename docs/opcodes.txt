Hydrogen Opcodes:

(0000) hlt - stops program

(0001) mov - move value into register

(0002) add - used for adding values
(0003) sub - used in subtracting values
(0004) mult - used in subtracting values
(0005) div - used in subtracting values
(0006) mod - used in subtracting values

(0007) inc - increment
(0008) dec - decriment

(0011) defi - this defines a new integer


jmp - jumps to place in file
(0020) call - calls the method and has the following operands as parameters

(0021) ret - returns value


(0030) dmp - prints the register and its values







Not yet implemented opcodes:
defb - this defines a new boolean (1 for true 0 for false)
defc - this defines a new char
cnst - defines new constant
coeq - compares the 2 values and checks if they are equal
cogt - compares the 2 values and checks if the 2nd value is greater than the first (less than operator)
colt - compares the 2 values and checks if the 1st value is greater than the first (greater than operator)
free - frees value from memory

Section opcodes:
sech - Header section
secs - Data section
secm - method section
